## 3. Интерфейс (UI / UX)

### 3.1 Схема экранов

```
[Главное меню]
      │
      ├─── [Настройки]
      │     
      ├─── [Рекорды]  
      │
      └─── [Игра] ──── [Пауза] ──── [Главное меню]
                │
                └─── [Game Over] ──── [Главное меню]
                                 └─── [Играть снова]
```

### 3.2 Главное меню

*Вайрфрейм  (Wireframe, схематичный набросок интерфейса) :*

```
┌──────────────────────────────┐
│        НАЗВАНИЕ ИГРЫ         │
│                              │
│        [ Играть     ]        │
│        [ Настройки  ]        │
│        [ Рекорды  ]        │
│        [ Выход      ]        │
│                              │
│   Рекорд: 0                  │
└──────────────────────────────┘
```

### 3.3 HUD (Heads-Up Display) — постоянно видимый игровой интерфейс

*Вайрфрейм:*

```
┌──────────────────────────────┐

│                              │
│        [игровая сцена]       │
│                              │
```
