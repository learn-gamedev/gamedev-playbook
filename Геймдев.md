# Геймдев

**Геймдев** (Game Development) — процесс создания компьютерных игр: от концепции до готового продукта. Включает [[Документация игрового проекта|проектирование]], программирование, арт, звук и тестирование.

---

## Категории компьютерных игр

### По жанру

| Жанр                        | Описание                                                                             | Примеры                        |
| --------------------------- | ------------------------------------------------------------------------------------ | ------------------------------ |
| **Action**                  | Акцент на реакции и моторике игрока                                                  | Doom, Hollow Knight            |
| **Adventure**               | Исследование мира, нарратив, головоломки                                             | Myst, The Forgotten City       |
| **RPG** (Role-Playing Game) | Прокачка персонажа, сюжетные выборы, глубокий сюжет, выбор в диалогах и открытый мир | Baldur's Gate, Undertale       |
| **Strategy**                | Планирование, управление ресурсами                                                   | StarCraft, Into the Breach     |
| **Simulation**              | Моделирование реальных систем                                                        | The Sims, Cities: Skylines     |
| **Puzzle**                  | Решение логических задач                                                             | Portal, Stephen's Sausage Roll |
| **Sports / Racing**         | Соревнование в реальных дисциплинах                                                  | FIFA, Trackmania               |
| **Horror**                  | Атмосфера страха, саспенс                                                            | Amnesia, Phasmophobia          |
| **Roguelike / Roguelite**   | Процедурная генерация, смерть = рестарт                                              | Hades, Dead Cells              |
| **Idle / Clicker**          | Пассивный прогресс, минимум действий                                                 | Cookie Clicker, Melvor Idle    |

### По перспективе и виду камеры

| Перспектива | Характеристика |
|-------------|----------------|
| **2D Side-scroller** | Камера сбоку, горизонтальное движение |
| **2D Top-down** | Камера сверху, вид с птичьего полёта |
| **Platformer** | 2D с гравитацией, прыжки по платформам |
| **First-Person (FPS)** | Вид от первого лица |
| **Third-Person** | Камера за спиной персонажа |
| **Isometric** | Псевдо-3D, проекция под углом 45° |

---

### По режиму игры

| Режим | Описание |
|-------|----------|
| **Single-player** | Один игрок против игрового мира |
| **Local co-op / PvP** | Несколько игроков на одном устройстве |
| **Online multiplayer** | Игроки соединяются через сеть |
| **MMO** | Тысячи игроков в одном постоянном мире |
| **Асинхронный мультиплеер** | Игроки не онлайн одновременно (ходы по очереди) |

### По темпу

| Темп | Описание | Примеры |
|------|----------|---------|
| **Real-time (RTS/Action)** | Всё происходит в реальном времени | StarCraft, Diablo |
| **Turn-based** | Игроки или ИИ ходят по очереди | Civilization, XCOM |
| **Real-time with pause** | RTS с возможностью поставить на паузу | Baldur's Gate 3 |

---

### По платформе

- **PC** (Windows / macOS / Linux) — самая открытая платформа для инди-разработки
- **Console** — требует сертификации и лицензий (Nintendo, PlayStation, Xbox)
- **Mobile** (iOS / Android) — монетизация через IAP / рекламу, приоритет touch-управления
- **Browser** — HTML5 / WebAssembly, минимальный порог входа для игрока
- **VR / AR** — особые требования к производительности и UX

## Типичные роли в команде

- **Game Designer** — механики, баланс, [[Документация игрового проекта|GDD]]
- **Programmer** — архитектура кода, физика, ИИ, [[Документация игрового проекта|TDD]]
- **Artist / Animator** — спрайты, UI, эффекты
- **Sound Designer** — музыка, SFX
- **Producer** — сроки, задачи, координация

> В небольшом проекте каждый член команды совмещает несколько ролей.
